### watch mode 와 개발 서버
프로그레스바 보기

    webpack --progress --colors

watch mode(자동 컴파일)

    webpack --progress --colors --watch

개발 서버 설치

    npm install webpack-dev-server -g

개발 서버 구동(watch mode 자동 실행)

    webpack-dev-server --progress --colors
